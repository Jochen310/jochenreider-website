Google Ads Asset Manager

Datengetriebene Kampagnenoptimierung mit der Google Ads API – entwickelt von Jochen Reider

Zum Tool

Google Ads Asset Manager Tool

Das Google Ads Asset Manager ist eine private Web-Applikation unter ads.jochenreider.com, die über die offizielle Google Ads API eine direkte Verwaltung und Optimierung von Google Ads-Kampagnen ermöglicht. Die Applikation wird ausschließlich für die Verwaltung des eigenen Google Ads-Kontos genutzt.

Kampagnen-Dashboard

Übersicht aller aktiven Kampagnen mit den wichtigsten Performance-Kennzahlen: Impressionen, Klicks, CTR, Kosten, Conversions und ROAS – direkt aus der Google Ads API.

Asset-Performance-Analyse

Detaillierte Ansicht aller Ad Assets (Headlines, Descriptions) einer Kampagne mit Google's Asset-Performance-Labels (BEST, GOOD, LOW) und Optimierungspotenzial.

KI-Optimierungsvorschläge

KI-generierte Vorschläge für verbesserte Headlines und Descriptions. Nutzer prüfen und genehmigen jeden Vorschlag explizit, bevor er via Google Ads API zurückgeschrieben wird.

Sichere Google OAuth Authentifizierung

Zugang ausschließlich per Google OAuth 2.0. OAuth-Token werden serverseitig verschlüsselt gespeichert. Kein externer Zugriff – nur der Kontoinhaber kann sich anmelden.

Technologie-Stack

  • Google Ads API (REST & gRPC)
  • Google OAuth 2.0
  • React & TypeScript
  • Node.js / Express
  • Anthropic Claude API (KI-Vorschläge)
  • Cloudflare Tunnel (HTTPS)

Datenschutz & Compliance: Die Applikation entspricht der Google API Services User Data Policy einschließlich der Limited Use Anforderungen. Google Ads-Daten werden nicht dauerhaft gespeichert und nicht an Dritte verkauft. Details in der Datenschutzerklärung und den Nutzungsbedingungen.

Google Ads Automatisierung & Reporting

Automatisiertes Kampagnen-Reporting

Mit einer direkten Anbindung an die Google Ads API erstelle ich vollautomatisierte Reporting-Pipelines, die Ihre Kampagnendaten in Echtzeit erfassen, aufbereiten und visualisieren. Schluss mit manuellem Export und stundenlangem Zusammenstellen von Berichten.

KPI-Dashboards

Individuelle Dashboards zeigen Ihnen die wichtigsten Kennzahlen auf einen Blick: Klickrate, Conversion-Rate, Cost-per-Acquisition, ROAS und mehr. Alle Daten werden automatisch aktualisiert und sind jederzeit abrufbar.

Performance-Analyse

Durch automatisierte Auswertungen erkennen Sie Trends und Optimierungspotenziale sofort. Die API-gesteuerte Analyse identifiziert unter- und überperformende Keywords, Anzeigengruppen und Kampagnen.

Alerting & Benachrichtigungen

Konfigurierbare Alerts informieren Sie automatisch, wenn Budget-Schwellenwerte erreicht werden, die Performance einbricht oder ungewöhnliche Aktivitäten erkannt werden – per E-Mail oder Slack.

Technologie-Stack

  • Google Ads API (REST & gRPC)
  • Python / Node.js
  • Google BigQuery & Looker Studio
  • Cloud Functions / Cloud Run
  • Automatisierte Scheduling-Pipelines
